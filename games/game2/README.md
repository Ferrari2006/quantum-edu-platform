## game2

当前整合的是旧仓库里的“Original Edition / easyver”版本（3→5 qubits、Fidelity、量子商店二次确认等）。

### 运行

在 `quantum_balatro_original/` 下安装依赖并启动：

```bash
pip install pygame qiskit qiskit-aer numpy
python display_engine.py
```

### 目录

- `quantum_balatro_original/`：`display_engine.py` + `game_state.py` + `quantum_backend.py`
